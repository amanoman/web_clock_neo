# 要求仕様：ストップウォッチ機能

## 概要

時計表示の下にストップウォッチ機能を追加する。
フォントサイズ・スタイルは既存の時計表示に揃える。

## 機能要件

- 経過時間を `MM:SS.xx`（分:秒.センチ秒）形式で表示する
- Start / Stop / Reset の3操作を提供する
  - Start: 計測開始（idle → running）
  - Stop: 一時停止（running → paused）
  - Reset: リセットして 00:00.00 に戻る（paused / running → idle）
- ページ再読み込みで状態はリセットされる（永続化なし）

## 非機能要件

- フォントサイズは既存の時計表示（`text-5xl md:text-8xl`）に揃える
- font-mono を使用してゼロ揃えを維持する
- 既存の時計の下に配置し、視覚的に分離する（セパレータまたは余白）

## 受け入れ条件

1. 初期表示は `00:00.00`
2. Start ボタン押下で計測開始
3. Stop ボタン押下で一時停止（経過時間を保持）
4. Reset ボタン押下で `00:00.00` に戻る
5. lint / test:ci / build がすべてパスする
